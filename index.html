Sankara

Transparence des prix alimentaires · Multi-ville
Food price transparency · Multi-city

Plateforme communautaire de transparence des prix, déployée à Ouagadougou, Galway et Les Ulis. Algorithme d'intégrité transparent (Sankara Integrity), détection automatique des bonnes affaires, flags expiré/survendu alignés sur le cadre BMCRF. Conformité RGPD intégrée dès la conception.

Community-driven price transparency platform, deployed in Ouagadougou, Galway, and Les Ulis. Transparent integrity algorithm (Sankara Integrity), auto-deal detection, expired/overpriced flags aligned with BMCRF consumer-protection framework. GDPR compliant by design.

React TypeScript Supabase Tesseract.js RGPD / GDPR

Hors-ligne d'abord

Offline first

Les applications que je conçois fonctionnent sans connexion. Les données synchronisent quand le réseau revient. Ce n'est pas une option — c'est une exigence d'infrastructure en Afrique de l'Ouest.

Apps I build work without connectivity. Data syncs when the network returns. This isn't an option — it's an infrastructure requirement in West Africa.

Souveraineté des données

Data sovereignty

Les données des utilisateurs restent dans leur juridiction. Inventaire complet des sous-traitants. Droits RGPD pleinement implémentés avant même d'être obligatoires.

User data stays in its jurisdiction. Full processor inventory. GDPR rights fully implemented before they're legally required.

Paiements locaux

Local payments

Intégration Orange Money, Moov Money (via CinetPay, Hub2). Les applications que je conçois acceptent les moyens de paiement que vos utilisateurs utilisent réellement.

Orange Money, Moov Money integration (via CinetPay, Hub2). Apps I build accept the payment methods your users actually use.
